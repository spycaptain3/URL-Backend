Backend task

A server that responds to mathematical operations sent via URL. it also maintains a history of the last 20 operations performed on the server.

How to Use the Program

Steps to Follow:

1. download and extract the files.

2. open terimal and the following command to install modules: npm i

3. run the index.js file using the command: node index.js

4. now, open browser and in address bar search: localhost:3000 opens the homepage

5. for every valid url expression: Json object is returned

6. localhost:3000/history : shows the last 20 operations performed with their answers.

7. for any invalid url : error 404 page is rendere
